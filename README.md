# Zero Motorcycles Home Assistant Integration for Home Assistant

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-orange.svg)](https://github.com/hacs/integration)


An  inofficial Home Assistant integration to monitor your Zero Motorcycle using the Zero NextGen API. This project has no affiliation with Zero Motorcycles.

---

## 🚀 Features

* **Real-time Telemetry:** Monitor battery state of charge (SoC), charging status, and odometer.
* **GPS Tracking:** Track your bike's last reported location on the Home Assistant map.
* **Automatic Discovery:** Automatically retrieves your motorcycle's `unit_number` using your app credentials.
